## 安装Ranger及其插件

```bash
brew install ranger # ranger主程序
brew install highlight # 预览代码高亮
brew install w3m # html页面预览
brew install atool # 存档预览
brew install poppler # PDF 预览
brew install mediainfo # 多媒体文件预览
brew install pandoc # 预览doc等office文件
```
